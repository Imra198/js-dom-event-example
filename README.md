# js-dom-event-example
A simple JavaScript project demonstrating DOM manipulation by updating a heading text in response to a button click.
The project uses vanilla JavaScript to access the DOM, handle events, and update content in response to user actions. It is intended as a beginner-friendly example of DOM manipulation and event handling.

## Technologies Used
- HTML5
- JavaScript (Vanilla)

## Features
- Dynamically changes the heading text when a button is clicked
- Uses getElementById and textContent for DOM manipulation
- Demonstrates basic event handling in JavaScript

- ## Project Structure
- index.html and Script.js

- ## How It Works
1. The page loads with a heading and a button.
2. JavaScript selects the heading and button elements from the DOM.
3. When the button is clicked, the heading text is updated dynamically.

## Purpose
This project was created as a beginner exercise to practice DOM manipulation, event handling, and interactive JavaScript behavior.

## How to Run
Open the `index.html` file in any modern web browser and click the button to see the heading text change.
